✨ Overview

NexWall X Ultimate is a modern Android wallpaper application designed to deliver a beautiful, fast, and immersive wallpaper browsing experience.

Built with modern Android technologies, NexWall X Ultimate provides users with access to high-quality wallpapers while maintaining performance, privacy, and usability.

⚙️ Installation

Clone the repository:

git clone https://github.com/rahman2008-svg/nexwall-x-ultimate.git

Open project:

cd nexwall-x-ultimate

Build Debug APK:

chmod +x gradlew
./gradlew assembleDebug

Build Release APK:

./gradlew assembleRelease
